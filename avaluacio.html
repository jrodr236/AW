Avaluació

  • L'avaluació de les diferents activitats es farà a través de graelles d'observació i/o de la correcció individual dels lliuraments de les tasques.
  • Les tasques s'hauran de lliurar utilitzant el moodle.
  • L'avaluació també seguirà els diferents resultats d'aprenantatge de la unitat formativa.

RA1. Instal·la, si s’escau, aplicacions web, descrivint les seves característiques i entorns d'ús.

Criteris d'avaluació

a. Descriu l’origen i evolució dels serveis web de darrera generació.
b. Instal·la si s’escau, diferents aplicacions web, descrivint les seves característiques i entorns d'ús.
c. Configura i utilitza el correu electrònic web . Gestiona comptes d’usuari.
d. Configura aplicacions de calendari web.
e. Reconeix les prestacions específiques de les aplicacions instal·lades (cites, tasques, entre d'altres).
f. Configura les aplicacions web per a integrar-les amb el servei de correu.
g. Cerca i interpreta documentació tècnica en les llengües oficials i en les de més ús al sector.
h. Documenta adequadament la configuració de les aplicacions web d'escriptori, les incidències aparegudes i les solucions aportades.

Continguts

1. Instal·lació, si s’escau, d’aplicacions web.
   1.1. Origen i evolució de les aplicacions web.
   1.2. Instal·lació si s’escau d’aplicacions web. Característiques i entorn d’ús.
   1.3. Correu electrònic web. Gestió de comptes d’usuari.
   1.4. Configuració de calendari web.
   1.5. Eines específiques web: cites, tasques, entre d’altres.
   1.6. Configuració de les diferents aplicacions per integrar-les a un servidor de correu.
   1.7. Documentació tècnica.
   1.8. Elaboració de documentacions.

RA2. Instal·la aplicacions d’ofimàtica web, descrivint les seves característiques i entorn d’ús.

Criteris d'avaluació

a. Cerca i interpreta l’origen i evolució de les aplicacions ofimàtiques.
b. Estableix la utilitat de les aplicacions de ofimàtica web.
c. Descriu diferents eines d’ofimàtica web (processador de textos, full de càlcul, entre altres).
d. Configura aplicacions d'ofimàtica web.
e. Gestiona els comptes de usuari.
f. Aplica criteris de seguretat en el accés dels usuaris.
g. Reconeix les prestacions específiques de cadascuna de les aplicacions instal·lades.
h. Utilitza les aplicacions de forma col·laborativa.
i. Documenta adequadament la configuració i gestió de les aplicacions ofimàtiques web, les incidències aparegudes i les solucions aportades.

Continguts

2. Realitza tasques de configuració d’aplicacions d’ofimàtica web.
   2.1. Origen i evolució de les aplicacions ofimàtiques web.
   2.2. Funcionalitats de les aplicacions d'ofimàtica web.
   2.3. Característiques distintives de les aplicacions d'ofimàtica web.
   2.4. Configuració i si s’escau instal·lació d’aplicacions d’ofimàtica web.
   2.5. Gestió d’usuaris.
   2.6. Permisos associats als usuaris. Comprovació de la seguretat.
   2.7. Utilització de les funcionalitats de les aplicacions d'ofimàtica web.
   2.8. Utilització col·laborativa de les aplicacions d'ofimàtica web.
   2.9. Documentació de la configuració, de les incidències aparegudes i de les solucions aportades.
